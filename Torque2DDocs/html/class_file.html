Sets the current position in the file.

You can set either a relative or absolute position to go to in the file.

File *foo;
... set up file ...
// Go to byte 32 in the file...
foo->setPosition(32);
// Now skip back 20 bytes...
foo->setPosition(-20, false);
// And forward 17...
